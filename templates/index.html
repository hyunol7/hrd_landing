자주 묻는 질문

Q 줌 화상교육 시간 편성이 어떻게 되나요? +

16시간 혼합교육 / 화상교육 8시간 신청자 : 9:00~18:00 (점심시간 1시간 포함)
8시간 혼합교육 : 9:00~14:00(오전) / 14:00~18:00(오후)

Q 집체교육 대신에 화상교육으로 실시할 수 있나요? +

네, 안전보건교육규정의 개정으로 가능합니다. (2023.2.21 개정)

Q 수료증 발급은 어디서 하나요? +

교육 100% 이수 시 당일 발급 가능합니다.
수료증 발급 경로: 내강의실 - [학습종료]에서 발급 가능합니다.

Q 관리감독자란 구체적으로 누구를 의미하나요? +

산업안전보건법 제16조에 따르면, 관리감독자란 "사업장의 생산과 관련되는 업무와 그 소속 직원을 직접 지휘·감독하는 직위에 있는 사람"을 의미합니다.
건설현장의 경우 주로 시공과 관련된 부서의 직·조·반장, 공사 과장·부장 등이 해당됩니다. (특별한 자격기준 없음)

Q 교육시간 전체를 우편교육이나 원격(동영상시청)교육으로 이수할 수 있나요? +

우편교육 또는 원격교육은 총 교육시간의 50%만 인정됩니다.
따라서, 전년도 무재해 사업장은 원격교육 50% + 화상 또는 집체교육 50%로 이수해야 합니다.
예) 화상교육 4시간 + 원격(동영상시청)교육 4시간

Q 우편교육 시험은 어떻게 응시하나요? +

pdf로 받아보시거나, 우편 교안을 받아서 시험을 응시하실 수 있습니다.

Q 관리감독자가 사업장 이동 시 교육을 다시 받아야 하나요? +

관리감독자 정기교육(16시간)은 부서가 이동되거나 소속 사업장의 근무 장소가 변경되더라도 기존에 받은 교육 시간이 인정됩니다.
다만, 부서 이동 또는 소속 사업장의 근무 장소 변경으로 인해 작업환경, 업무 내용 등이 변경된 경우에는 산업안전보건법 제29조 제2항에 따라 작업내용 변경 시 교육(2시간)을 이수하여야 합니다.

Q 줌 이용 매뉴얼이 있나요? +

네, 줌 사용법 안내 자료를 제공하고 있습니다.
홈페이지 공지사항에서 "줌 사용법" 또는 "화상교육 안내"를 검색하시면 상세한 매뉴얼을 확인하실 수 있습니다.
