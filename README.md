# 대한안전보건교육원 스타일 랜딩페이지

[대한안전보건교육원](https://kshec.co.kr) 공식 사이트의 모든 기능과 콘텐츠를 구현한 랜딩페이지입니다.

## 🎯 주요 기능

### 1. 완전한 페이지 구성
- ✅ 최상단 빠른 메뉴 (화상줌입장, 단체수강신청, 수료증출력 등)
- ✅ 실시간 문자상담 배너 (02-867-6404)
- ✅ 메인 히어로 섹션 ("아.직.도 교육장에 직접 방문하시나요?")
- ✅ 과태료 경고 안내 (최대 500만원)
- ✅ 빠른 서비스 섹션 (5가지 서비스)
- ✅ 교육 방식 안내 (화상/우편/집체/혼합)
- ✅ 교육 과정 상세 안내 (관리감독자/근로자/4대법정의무)
- ✅ 확장된 FAQ (8개 질문)
- ✅ 관련 기관 링크 (7개 기관)

### 2. 사이드 팝업 상담 폼
- 우측 하단 고정 "빠른상담 문의" 버튼
- 클릭 시 우측에서 슬라이드되는 상담 폼
- **이중 저장**: DB + 구글 시트

### 3. 상담 폼 필드
- **기업명** (필수)
- **연락처** (필수, 자동 포맷팅: 010-0000-0000)
- **담당자 성함** (필수)
- **문의사항** (선택)
- **개인정보 수집 동의** (필수)

### 4. 데이터 저장
- **SQLite 데이터베이스**: 로컬 저장
- **구글 시트**: 클라우드 백업 및 실시간 확인
- 자동으로 테이블 생성
- 상담 문의 내역 저장 및 조회

## 🚀 설치 및 실행

### 1. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 구글 시트 연동 설정 (선택사항)
자세한 내용은 `GOOGLE_SHEETS_SETUP.md` 참조

간단 요약:
1. 구글 시트 생성
2. Apps Script 코드 배포
3. 웹 앱 URL을 `static/js/script.js`에 입력

### 3. 서버 실행
```bash
python app.py
```

### 4. 브라우저에서 접속
```
http://localhost:5001
```

## API 엔드포인트

### 상담 문의 저장
- **POST** `/consultation`
- 요청 본문 (JSON):
```json
{
  "company_name": "회사명",
  "contact_number": "010-1234-5678",
  "manager_name": "홍길동",
  "inquiry_content": "문의내용"
}
```

### 전체 상담 문의 조회
- **GET** `/consultations`
- 응답:
```json
{
  "status": "success",
  "count": 10,
  "data": [...]
}
```

## 데이터베이스 구조

### Consultation 테이블
| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| id | Integer | 자동 증가 ID (PK) |
| company_name | String(200) | 기업명 |
| contact_number | String(50) | 연락처 |
| manager_name | String(100) | 담당자명 |
| inquiry_content | Text | 문의내용 |
| created_at | DateTime | 접수시간 |

### 실시간 문자상담 배너
- 전화번호: 02-867-6404
- 주황색 그라데이션 배경

### 메인 히어로 섹션
- "아.직.도 교육장에 직접 방문하시나요?"
- "시간과 돈을 SAVE 하는 법"
- 3가지 특징: easy(쉽고), one-day(빠르게), one-page(한번에 해결)

### 과태료 경고
- "금년도 '관리감독자 교육' 실시하셨나요?"
- 미이수 시 과태료 최대 500만원
- 1차 50만원, 2차 250만원, 3차 500만원

### 빠른 서비스 (5개)
1. 법정교육 대상조회
2. 견적서 자동발송
3. 교육후기
4. 안전자료 무료다운
5. 단체신청

### 교육 방식 (4가지)
1. **화상교육**: Zoom 실시간 온라인
2. **우편교육**: 교안 수령 후 자가학습
3. **집체교육**: 교육장 방문 대면교육
4. **혼합교육**: 화상+우편, 화상+원격 조합

### 교육 과정 (3가지)
1. **관리감독자 정기교육**: 16시간 (연간)
2. **근로자 정기교육**: 분기당 6시간
3. **4대 법정의무교육**: 연 1회

### FAQ (8개 질문)
- 줌 화상교육 시간 편성
- 집체교육 대신 화상교육 가능 여부
- 수료증 발급 방법

## 💡 주요 기능 설명

### 1. 전화번호 자동 포맷팅
입력하는 동안 자동으로 하이픈(-) 추가
예: 01012345678 → 010-1234-5678

### 2. 이중 데이터 저장
- **SQLite DB**: 로컬 서버에 영구 저장
- **구글 시트**: 클라우드에 실시간 백업

### 3. 팝업 제어
- 우측 상단 X 버튼 클릭
- 오버레이 클릭
- ESC 키 누르기

### 4. 알림 메시지
- 성공 시: 초록색 알림
- 실패 시: 빨간색 알림
- 5초 후 자동 사라짐

### 5. 반응형 디자인
- 모바일, 태블릿, 데스크톱 모두 지원
- 화면 크기에 따라 자동 조정

### 6. 스크롤 애니메이션
- 섹션별 부드러운 페이드인 효과
- 카드 호버 시 3D 효과

## 참고사항

- 개발 서버로 실행되므로 실제 운영 환경에서는 WSGI 서버 사용 권장
- 데이터베이스는 SQLite를 사용하며, 필요시 PostgreSQL 등으로 변경 가능
- SECRET_KEY는 실제 운영 환경에서 반드시 변경할 것
